# Anurag Nimkande - Portfolio Website

A modern, responsive portfolio website for Anurag Nimkande, a Computer Engineering student and software developer.

## Features

- **Dark/Light Mode Toggle** - User preference saved in local storage
- **Responsive Design** - Works on all devices from mobile to desktop
- **Smooth Animations** - Fade-in effects and smooth scrolling
- **Interactive Elements** - Form submission, navigation, theme toggle
- **Modern UI** - Clean, professional design with code-inspired elements
